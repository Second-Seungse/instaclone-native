# .expo-shared
> .expo-shared 폴더는 프로젝트의 모든 개발자와 공유할 상태를 생성하는 명령을 실행할 때 생성됩니다.
예를 들어 "npx expo-optimize"가 있습니다.

> "assets.json" 파일은 "expo-optimize"를 통해 최적화되었으며 다시 처리할 필요가 없는 자산에 대해 설명합니다.

> 공동작업자와 ".expo-shared" 폴더를 공유해야 합니다.
